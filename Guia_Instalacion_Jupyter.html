Guía de Instalación

Jupyter Notebook / Jupyter Lab

Para el curso de Data Analytics con Pronósticos en Python

1 Opción Recomendada: Usar Anaconda (Más Fácil)

Recomendado para principiantes.

  1. Descarga Anaconda desde: https://www.anaconda.com/download
  2. Instala según tu sistema operativo (Windows / macOS / Linux).
  3. Abre el **Anaconda Prompt** (en Windows) o Terminal.

2 Crear Entorno Virtual

conda create -n pronosticos python=3.11 -y

Activar el entorno:

conda activate pronosticos

3 Instalar Jupyter Notebook / Lab

conda install jupyterlab -y

O con pip:

pip install jupyterlab notebook

4 Instalar Bibliotecas del Curso

pip install pandas numpy matplotlib seaborn statsmodels prophet scikit-learn plotly pmdarima

5 Ejecutar Jupyter

jupyter lab

O para la versión clásica:

jupyter notebook

Problemas Comunes y Soluciones

  • Comando no encontrado: Asegúrate de tener activado el entorno (conda activate pronosticos).
  • Permisos en Windows: Ejecuta Anaconda Prompt como Administrador.
  • Instalación lenta: Usa -y para aceptar automáticamente.
